☁️ Sync across devices

Connect a free Supabase project to keep your saves in sync on every device.

1. Create a free project at supabase.com

2. Open SQL Editor and run:

create table bookmarks (
  id text primary key,
  user_id uuid references auth.users not null,
  recipe jsonb not null,
  saved_at timestamptz default now()
);
alter table bookmarks enable row level security;
create policy "own" on bookmarks
  for all using (auth.uid() = user_id)
  with check (auth.uid() = user_id);

create table shopping_lists (
  id text primary key,
  user_id uuid references auth.users not null,
  recipe_name text not null,
  items jsonb not null,
  saved_at timestamptz default now()
);
alter table shopping_lists enable row level security;
create policy "own" on shopping_lists
  for all using (auth.uid() = user_id)
  with check (auth.uid() = user_id);

3. In Authentication → URL Configuration, add your app URL to Redirect URLs

4. In Settings → API, copy your Project URL and anon key below

Sign in to start syncing. We'll send a magic link — no password needed.
